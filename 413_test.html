Student Information > Bulk Student Import

  • You can download the student import template from the link provided below.
  • All columns that are marked with * are required fields which must be entered. Click here to know the details of each columns and what values can be entered.
  • None of the fields can have value over 255 characters.
  • After successful import, you will be shown both success count and error count. If error count is more than 0, an error file will be downloaded. It will have list of all the records which couldn't be read because of some error, you can find the reason for the error in column “Error Message”. After correcting the records you can use the same file for import again.
  • Note: This import function only supports csv files(comma separated). So when you are saving this file from any spreadsheet software, please save it as csv file.
  • If you have more confusion or if you face any problem which you cannot resolve, please feel free to contact us at contact@edutinker.com
